# TODOアプリ
TODOタスクを管理できます。<br>
入力したTODOは完了させたり、未完了に戻したり、削除したりできます。<br>
データの永続化はしておりません。<br>

[https://kazuki-takane.github.io/my-todo/](https://kazuki-takane.github.io/my-todo/)

## 機能一覧
- TODO追加機能
- TODO完了・未完了管理機能
- TODO削除機能

## 使用技術
- React 18.2.0
- TypeScript 4.9.5
- styled-components 6.0.3
